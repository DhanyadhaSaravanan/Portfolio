# Personal Portfolio Website

A modern, responsive single-page portfolio built with React (Vite), Tailwind CSS, and React Icons.

## Features

- Sticky navigation with active section highlighting
- Smooth scrolling and fade-in animations on scroll
- Responsive design for mobile, tablet, and desktop
- Skills with animated progress bars
- Project, certification, and achievement cards
- Contact form with frontend validation
- Back-to-top button

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Customization

Edit the placeholder content in `src/data/`:

- `profile.js` — name, role, about, education, social links
- `skills.js` — skill categories and proficiency levels
- `projects.js` — project cards
- `certifications.js` — certificates
- `achievements.js` — achievements

## Tech Stack

- React 19 + Vite
- Tailwind CSS v4
- React Icons
